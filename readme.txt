Ceci est le repository de mon projet Health North.

Health North est une entreprise médicale qui requiert une application Web et une application mobile qui communiqueront toutes les deux avec une API afin d'interagir avec une BDD.

Le service informatique externe permettra aux patients prendre et consulter leurs réservations et services connexes, d'ajouter des alarmes de prise de médicament dans leur application mobile, de se créer un compte et de consulter leur dossier médical.

Techno utilisées : Draw.io, DbDiagram.io, PHP, Html, CSS, Doctrine ORM, Framework Symfony, XAMPP, Vscode, Git, Github, Bootstrap, Postman, Figma, MySQL, Nelmio bundle, JWT authentication, Ficture Bundle
